## 练习题使用指南

| 题型 | 说明 | 建议用时 |
|------|------|:-------:|
| 🔧 代码修改题 | 修改参数/代码，观察并解释变化 | 15-30分钟 |
| ✏️ 实现题 | 从零实现简化版算法，理解核心逻辑 | 30-60分钟 |
| 💡 分析题 | 解释算法原理、适用场景和局限性 | 10-20分钟 |
| ⚖️ 对比题 | 对比不同算法的优缺点和适用条件 | 15-25分钟 |
| 🎯 应用题 | 针对特定场景选择算法并说明理由 | 10-20分钟 |

**建议做法：**

1. 先独立思考，再查看参考答案提示
2. 代码题务必动手运行，观察实际结果
3. 分析题和对比题尝试用自己的话表述
4. 每完成一个模块的练习，对照阶段自检清单检验

## 参考答案提示

> 以下仅为思路指引，非完整答案。请先独立思考再参考。

### 模块00 提示

**练习1**：余弦相似度衡量方向而非大小，[1,1,0] 和 [2,2,0] 方向相同但大小不同。欧氏距离衡量绝对位置差异。

**练习2**：曼哈顿距离 = 各维度差值绝对值之和；切比雪夫距离 = 各维度差值绝对值的最大值；余弦距离 = 1 - (A·B)/(||A||·||B||)。

**练习3**：数据理解帮助判断数据质量和可用性，跳过可能导致在脏数据上做无效准备。业务理解偏差导致整个挖掘方向错误。循环是因为后续步骤可能发现需要回退修正。

**练习4**：1→推荐/关联规则 2→异常检测 3→关联规则。关键看目标：预测标签→分类/回归，发现模式→聚类/关联，发现异常→异常检测。

### 模块01 提示

**练习5**：OLTP 追求事务处理速度和一致性，OLAP 追求查询分析性能。不可更新保证历史可追溯。雪花模型更规范但查询更复杂，适合维度层次深的场景。

**练习6**：关注 ETL 的清洗步骤如何处理缺失值，维度扩展后外键关系的变化，聚合粒度改变后度量值的差异。

**练习7**：上卷=汇总（如月→年），下钻=细化（如年→月），切片=固定一个维度值，切块=选定维度范围，旋转=交换行列。

**练习8**：维度表：城市、品类、时间；事实表：销售额。用切片+下钻。对比需旋转或并列。

### 模块02 提示

**练习9**：MinMaxScaler 将数据压缩到 [0,1]，StandardScaler 使均值为 0 标准差为 1。偏态数据中位数比均值更稳健。2σ 比三 σ 更严格，标记更多异常。

**练习10**：方差阈值法遍历每列计算方差，删除低于阈值的列。相关性过滤计算每列与目标列的皮尔逊相关系数。

**练习11**：对数变换压缩长尾使分布更接近正态。特征交叉捕捉交互效应。决策树类算法不需要标准化，因为基于特征比较。

**练习12**：1→散点图 2→柱状图/条形图 3→折线图 4→散点矩阵/平行坐标 5→箱线图。

### 模块03 提示

**练习13**：alpha 太小→欠正则化→过拟合，alpha 太大→过正则化→欠拟合。去掉特征看系数变化判断特征重要性。L1 正则化会将无关特征系数压缩为 0。

**练习14**：梯度 = X^T(Xw-y)/n，更新 w = w - lr * gradient。注意特征标准化可加速收敛。

**练习15**：线性回归预测连续值，逻辑回归预测概率（0-1）。逻辑回归用 Sigmoid 将输出映射到概率，损失函数为交叉熵。

**练习16**：1→线性回归 2→逻辑回归 3→线性回归 4→逻辑回归 5→线性回归。关键看目标是连续值还是类别。

### 模块04 提示

**练习17**：k 太小→过拟合（噪声影响大），k 太大→欠拟合（边界过于平滑）。标准化对 KNN 至关重要，因为 KNN 基于距离计算。

**练习18**：fit 阶段计算每个类别的先验概率和各特征的均值/方差。predict 阶段用贝叶斯公式计算后验概率取最大值。

**练习19**：信息增益偏向多值特征因为分得越细信息增益越大。增益率用分裂信息做归一化。预剪枝简单但可能过早停止，后剪枝更准确但计算量大。

**练习20**：KNN 预测慢（需计算所有距离），朴素贝叶斯快（查表概率）。KNN 对量纲敏感，决策树不敏感。SVM 假设最大间隔，朴素贝叶斯假设条件独立。

**练习21**：1→朴素贝叶斯（特征独立假设+高维稀疏）2→决策树（可解释）3→SVM（高维有效）4→半监督学习。

### 模块05 提示

**练习22**：不平衡时准确率不可靠，应看 F1 或 AUC。折数越多估计越稳定但计算量越大。AUC=0.5 等于随机猜测，AUC=1.0 是完美分类。

**练习23**：精确率重要→垃圾邮件（不想误判正常邮件），召回率重要→癌症检测（不想漏诊）。F1 是精确率和召回率的调和平均。

**练习24**：SMOTE 在少数类样本间插值生成新样本，比复制更有多样性。LIME 局部近似，SHAP 基于博弈论全局一致。PDP 展示特征对预测的平均影响，SHAP 展示每个样本的贡献。

**练习25**：1→F1/AUC + SMOTE 2→召回率优先 + 降低阈值 3→AUC/NDCG + 位置去偏。

### 模块06 提示

**练习26**：集成方法通常优于单模型。n_estimators 增大收益递减。learning_rate 太大容易过拟合，太小需要更多基学习器。

**练习27**：Bagging 并行训练降低方差（如随机森林），Boosting 串行训练降低偏差（如 AdaBoost/GBDT），Stacking 用异质学习器+元学习器。

**练习28**：LightGBM 将连续特征离散化为直方图加速分裂查找。CatBoost 用有序目标编码处理类别特征避免目标泄露。大数据选 LightGBM，类别特征多选 CatBoost，精度优先选 XGBoost。

**练习29**：1→LightGBM 2→CatBoost 3→XGBoost 4→Stacking。

### 模块07 提示

**练习30**：核心循环：分配每个点到最近质心→重新计算质心→重复直到收敛。注意初始化对结果的影响。

**练习31**：肘部法则看拐点，轮廓系数看最大值。KMeans 对非球形簇效果差，DBSCAN 能处理任意形状。eps 控制邻域大小，min_samples 控制核心点密度。

**练习32**：支持度衡量频率，置信度衡量条件概率，提升度衡量是否真正相关（>1 才有意义）。先验性质：频繁项集的子集也频繁。FP-Growth 用 FP 树避免候选项集生成。

**练习33**：PCA 中心化后做特征分解，SVD 直接对矩阵分解。SVD 可处理缺失值（推荐系统），PCA 不能。推荐系统用 SVD 因为用户-物品矩阵稀疏。

**练习34**：1→聚类（KMeans/层次聚类）2→关联规则（Apriori/FP-Growth）3→PCA/t-SNE 4→异常检测（孤立森林/LOF）。

### 模块08 提示

**练习35**：节点少→欠拟合，节点多→过拟合。Dropout 防止过拟合。Sigmoid 有梯度消失问题，训练比 ReLU 慢。

**练习36**：前向传播：z1=W1·x+b1, a1=ReLU(z1), z2=W2·a1+b2, y=Softmax(z2)。反向传播：从输出层向输入层逐层计算梯度。数值梯度验证是调试反向传播的标准方法。

**练习37**：自编码器有非线性激活，PCA 是线性的。VAE 让编码器输出分布参数而非确定值，重参数化让采样操作可微。Q=查询，K=键，V=值，注意力 = softmax(QK^T/√d)V。Transformer 全连接可并行，RNN 顺序依赖。

**练习38**：CNN 局部感受野+权值共享，RNN 时序递归，Transformer 全局自注意力。CNN 和 Transformer 可并行，RNN 不行。Transformer 复杂度 O(n²)，RNN 是 O(n)。

### 模块09 提示

**练习39**：1→协同过滤/矩阵分解 + NDCG 2→ARIMA/LSTM + RMSE 3→社区发现（Louvain/谱聚类）+ 模块度 4→TF-IDF + 分类器 + F1 5→孤立森林/流式检测 + 延迟/召回率。

**练习40**：联邦学习让数据不出本地，权衡是通信开销和精度损失。差分隐私加噪声保护个体，精度略降。医疗、金融等隐私敏感场景必须用联邦学习。

**练习41**：NLP→文本→TF-IDF/BERT→语义理解；时序→序列→ARIMA/LSTM→平稳性/概念漂移；推荐→矩阵→协同过滤→冷启动/稀疏；图→图结构→PageRank/GNN→可扩展性；流数据→无限流→滑动窗口→概念漂移。

**练习42**：中文分词质量直接影响特征质量。停用词过滤减少噪声特征。TF-IDF 比词袋模型更好地区分词的重要性。
