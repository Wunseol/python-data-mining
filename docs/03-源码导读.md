以下以KNN分类为例，展示典型运行时各函数的调用顺序：

```mermaid
sequenceDiagram
    participant User as 用户
    participant Main as __main__
    participant File2Mat as file2matrix
    participant AutoNorm as autoNorm
    participant Classify0 as classify0
    participant DatingTest as datingClassTest
    participant Img2Vec as img2vector
    participant Handwriting as handwritingClassTest
    participant PlotDB as plot_decision_boundary

    User->>Main: python K近邻算法.py

    alt 约会网站分类
        Main->>File2Mat: file2matrix("datingTestSet2.txt")
        File2Mat-->>Main: 返回特征矩阵和标签
        Main->>AutoNorm: autoNorm(dataSet)
        AutoNorm-->>Main: 返回归一化数据、范围、最小值
        Main->>DatingTest: datingClassTest()
        DatingTest->>File2Mat: file2matrix(训练集)
        DatingTest->>AutoNorm: autoNorm(训练集)
        DatingTest->>Classify0: classify0(测试样本, 训练集, 标签, k=3)
        Note over Classify0: 1. 计算欧氏距离<br/>2. 选取K个最近邻<br/>3. 多数投票分类
        Classify0-->>DatingTest: 返回分类结果
        DatingTest-->>Main: 输出错误率
    end

    alt 手写数字识别
        Main->>Handwriting: handwritingClassTest()
        Handwriting->>Img2Vec: img2vector(图像文件)
        Img2Vec-->>Handwriting: 返回1x1024向量
        Handwriting->>Classify0: classify0(测试向量, 训练矩阵, 标签, k=3)
        Classify0-->>Handwriting: 返回分类结果
        Handwriting-->>Main: 输出错误率
    end

    alt 决策边界可视化
        Main->>PlotDB: plot_decision_boundary()
        PlotDB->>Classify0: classify0(网格点, 训练集, 标签, k)
        Classify0-->>PlotDB: 返回各点分类
        PlotDB-->>User: 显示决策边界图
    end
```
